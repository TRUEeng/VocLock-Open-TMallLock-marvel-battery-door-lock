# VocLock-Open-TMallLock-marvel-battery-door-lock
This project is built upon the foundation of VocLock door lock system. It streamlines the unlocking process of the door lock. The currently tested lock model is TMallLock (Tmall). Through conventional reverse engineering of the Tmall Campus application, I have successfully extracted and refined the unlocking process. Users can now operate without being affected by those annoying advertisements and sluggish software performance.

I believe this raises no issues, as this project itself does not involve any cracking or exploitation. All keys cannot be obtained through any secondary means — they can only be acquired through legitimate methods.
